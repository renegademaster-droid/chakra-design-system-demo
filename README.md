# Design System

A public design system built with **Chakra UI** and **Figma** design tokens. Includes a component library, token-driven theme, and a documentation site with dark mode support.

## Stack

- **React** + **TypeScript** + **Vite**
- **Chakra UI** for components and theming
- **Figma tokens** (`figma-tokens.ts`) mapped into the theme via `figma-to-chakra.ts`

## Getting started

```bash
npm install
npm run dev
